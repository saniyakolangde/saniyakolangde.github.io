​This was the capstone project delivered for the Samsung Innovation Campus AI Course. It involved image processing of a vast skin lesion image dataset and building a deep learning neural network to classify these images into the multi-class labels. Alongside, creation and training of the model, the team developed a web app for the model to allow images to be uploaded for classification and prediction results with the model deployed.
